    Pertanyaan yang sering diajukan

    Bisakah dipakai offline?

    Ya. Instal sebagai PWA di perangkatmu dan catatan akan tersimpan di penyimpanan lokal.

    Di mana data saya disimpan?

    Catatan tersimpan di browser kamu. Jika login ke backend pilihanmu, semua data dienkripsi end-to-end.

    Bagaimana cara ekspor?

    Buka catatan dan klik "Ekspor" untuk mengunduh file Markdown atau langsung salin ke clipboard.

    Privasi & keamanan

    • Catatan disimpan di perangkatmu menggunakan enkripsi Web Crypto.
    • Baca kebijakan privasi lengkap di bagian pengaturan aplikasi.
    • Dukung login anonim atau Google sesuai preferensi pengguna.

    Dibuat dengan semangat belajar oleh Abelion. Instal ke layar utama untuk pengalaman terbaik.
