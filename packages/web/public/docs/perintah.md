
Berikut adalah referensi cepat untuk semua perintah yang tersedia di Lembaran CLI.

| Perintah | Fungsi | Deskripsi |
| :--- | :--- | :--- |
| `mulai` | **Menu Utama** | Membuka antarmuka interaktif (TUI) |
| `pantau` | **Status** | Memeriksa kesehatan database dan jumlah catatan |
| `jelajah` | **Search** | Mencari catatan menggunakan algoritma Fuzzy Search |
| `ukir` | **Editor** | Membuat catatan baru atau mengedit yang sudah ada |
| `tanam` | **Import** | Mengimpor file Markdown (.md) ke dalam brankas |
| `petik` | **Export** | Mengekspor seluruh brankas ke format `.lembaran` terenkripsi |
| `layani` | **Server** | Menjalankan API Server lokal untuk integrasi pihak ketiga |
| `keluar` | **Exit** | Menutup aplikasi CLI |

## Tips Penggunaan

- **Pencarian Cepat**: Gunakan `jelajah <kata-kunci>` untuk langsung menyaring hasil.
- **Navigasi TUI**: Gunakan tombol **Panah (↑/↓)** untuk berpindah menu dan **ENTER** untuk memilih.
- **Keamanan**: CLI akan meminta kata sandi brankas setiap kali Anda mencoba mengakses data sensitif jika sesi belum terbuka.
