
Selamat datang di ekosistem **Lembaran**. Panduan ini akan membantu Anda memahami dasar-dasar kedaulatan data dan bagaimana memulai arsip digital Anda sendiri dalam hitungan menit.

## 1. Langkah Pertama

Buka Brankas Anda melalui tombol **"Buka Brankas"** di pojok kanan atas. Jika ini pertama kalinya, Anda akan diminta untuk menetapkan **Frasa Sandi Utama**.

> [!WARNING]
> Frasa sandi Anda adalah kunci tunggal untuk mendekripsi data. Lembaran tidak menyimpan sandi Anda di server mana pun. Jika hilang, data Anda tidak dapat dipulihkan.

## 2. Keamanan Sentinel

Setiap aksara yang Anda tulis akan melewati protokol enkripsi **AES-GCM 256-bit** secara lokal di perangkat Anda. Data yang disimpan di database (IndexedDB) dalam keadaan terenkripsi penuh.

## 3. Ekosistem Shell

Bagi Anda yang menyukai kecepatan, Lembaran hadir dengan antarmuka terminal yang kuat. Anda dapat mengelola arsip tanpa pernah menyentuh mouse.

```bash
bun install -g Abelion512/lembaran
```

## 4. Filosofi Berdikari

Lembaran bukan sekadar aplikasi catatan; ini adalah manifesto tentang kepemilikan data. Kami menggunakan standar terbuka dan memastikan Anda dapat mengekspor seluruh data Anda kapan saja dalam format yang dapat dibaca mesin.
