## ğŸ› ï¸ Mulai Menggunakan / *Getting Started*

### Prasyarat / *Prerequisites*
Pastikan Anda telah menginstal [Bun](https://bun.sh/).

### Instalasi / *Installation*
```bash
bun install
```

### Pengembangan / *Development*

