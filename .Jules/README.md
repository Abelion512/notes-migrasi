Folder ini berisi spesifikasi, kapabilitas, dan panduan untuk agen kecerdasan buatan (Jules) yang bekerja pada repositori ini.

## Isi Folder:
- `SPEC.md`: Aturan operasional, identitas, dan batasan Jules.
- `MCP_CAPABILITIES.md`: Daftar teknis alat eksternal (MCP) yang dapat digunakan Jules.
- `INTEGRATION_SCENARIOS.md`: Contoh penggunaan nyata Jules dalam proyek ini.
- `TEMPLATE.md`: Template standar untuk diterapkan di repositori lain.
- `palette.md`: Brand identity and icon system documentation (Added by User).

## Cara Menggunakan Template:
1. Salin folder `.Jules/` ke repositori baru.
2. Edit `TEMPLATE.md` dan ubah namanya menjadi `SPEC.md`.
3. Perbarui `AGENTS.md` di repositori tersebut agar merujuk ke folder ini.
